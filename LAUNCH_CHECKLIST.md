## Common Issues & Solutions

### Form Not Submitting
**Problem:** Form submits but nothing happens  
**Solution:** 
- Check browser console (F12)
- Use Formspree (formspree.io) to capture forms
- Or build backend API

### Mobile Menu Not Working
**Problem:** Hamburger menu doesn't toggle  
**Solution:**
- Ensure script.js is loading
- Check browser console for errors
- Verify all files in same folder

### Animations Not Smooth
**Problem:** Animations lag or stutter  
**Solution:**
- Reduce animation complexity
- Optimize images
- Check browser performance
- Test on different device

### Slow Loading
**Problem:** Website takes too long to load  
**Solution:**
- Compress images with TinyPNG
- Minify CSS/JS
- Use CDN (Netlify/Vercel do this)
- Check server performance

### Contact Info Not Updating
**Problem:** Changed number but still shows old  
**Solution:**
- Ctrl+Shift+Delete (clear cache)
- Hard refresh (Ctrl+F5)
- Check all instances in HTML
- Search & replace to be sure
